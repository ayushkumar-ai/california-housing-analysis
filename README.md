# California Housing Price Analysis

## Project Overview
This project performs exploratory data analysis (EDA) on the California Housing dataset using Python.

The goal is to understand data distribution, handle missing values, and analyze patterns affecting house prices.



## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn



## Key Analysis Performed
- Checked dataset structure and statistics
- Handled missing values in total_bedrooms
- Visualized median_house_value distribution
- Observed price capping at 500000
- Explored correlations between features



## Concepts Applied
- Data Cleaning
- Data Visualization
- Exploratory Data Analysis (EDA)
- Basic Regression Understanding
